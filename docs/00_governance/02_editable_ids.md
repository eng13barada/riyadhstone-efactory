# Editable IDs (Client Click-to-Request-Change)

## Principle

Every editable text/media must have a stable ID that never changes.

## ID Format

[BRAND].[PAGE].[SECTION].[COMPONENT].[FIELD]

## Examples

RSW.HOME.HERO.CTA.PRIMARY_TEXT
RSW.HOME.HERO.CTA.PRIMARY_LINK
RSW.PRODUCTS.CARD.RIYADEX.TITLE
RSW.PRODUCTS.CARD.RIYADEX.MEDIA_THUMB

## Notes

- IDs are NOT translated.
- UI languages change, IDs remain stable.
